# .agents/AGENTS.md

`.agents/` contains local agent configuration, prompts, or temporary agent-facing notes.

## Rules

- Do not place runtime code here.
- Do not place generated exports here.
- Do not place reference documents here.
- Do not place templates here.
- Do not place tests here.

Project-wide rules belong in root `AGENTS.md`.

Durable documentation belongs in `docs/`.
