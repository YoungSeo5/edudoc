# references/AGENTS.md

`references/` stores raw reference and source documents.

## Responsibility

Use this folder for reference material that edudoc may learn from or compare against.

## Rules

- Files here are reference material and example forms from which template
  candidates may be extracted — not generated outputs.
- Do not modify original reference files unless explicitly asked.
- Do not store runtime exports here.
- Do not store tests here.
- Do not claim a reference is official unless verified.
- A reference's text describing its own fonts/margins is evidence, not a parsed
  style value; do not present it as extracted style.
- Do not create empty future folders unless needed.
